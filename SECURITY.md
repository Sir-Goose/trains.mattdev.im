# Security Policy

## Reporting a Vulnerability

Please report security issues privately by email: **security@trains.mattdev.im**.

Include:
- affected endpoint/file
- reproduction steps
- impact assessment

We will acknowledge receipt and follow up with remediation steps.

## Active Secret Leaks

If you discover an active key/token leak, **do not open a public GitHub issue**.
Report it privately via the email above so credentials can be rotated immediately.
